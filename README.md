# ft_vox

42Tokyoの課題として制作した、C++17 / OpenGL 4.1 製のMinecraft風ボクセルエンジンです。

---

## 操作方法

| キー / 入力                | 操作                                              |
| -------------------------- | ------------------------------------------------- |
| `W` `A` `S` `D`            | 移動                                              |
| マウス                     | 視点操作                                          |
| `Space`                    | ジャンプ（地上） / 上昇（水中・飛行モード）       |
| `Left Ctrl`                | 沈む（水中） / 下降（飛行モード）                 |
| `Left Shift`               | 高速モード切替（通常速度の20倍）                  |
| `Space` ×2（ダブルタップ） | 飛行モード / 重力モード切替                       |
| `左クリック`               | ブロックを壊す                                    |
| `右クリック`               | ブロックを置く                                    |
| `1` 〜 `6`                 | ブロック種別選択（石 / 草 / 土 / 砂 / 雪 / 木材） |
| `Esc`                      | カーソル解放                                      |

---

## ビルド・起動方法

**動作要件**

- CMake 3.24以上
- C++17対応コンパイラ
- OpenGL 4.1（macOS: AppleドライバによるMetal経由のOpenGL）

**コマンド**

```bash
# ビルド
make

# ビルド＆起動（ランダムシード）
make run

# ビルド＆起動（シード指定）
make run 12345
```
